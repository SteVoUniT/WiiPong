# Nintendo Wii PowerPC Pong Project
A demonstration of C/C++ running on the Nintendo Wii using GRRLIB libraries

# Built With
- C / C++
- devkitPro
- GRRLIB
- Nintendo Wii (PowerPC)

# Build
Requires devkitPro toolchain and GRRLIB.

# What Does This Prove?
- Cross-compiling C/C++ for PowerPC
- Graphics rendering using GRRLIB
- Input handling on console hardware
- Basic game loop and state management
